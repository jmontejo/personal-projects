Parámetros

Los diferentes parámetros controlan los detalles del proceso de cálculo. Todos los parámetros son numéricos, y algunos tienen restricciones debiendo ser enteros y/o positivos. Si algún parámetro tiene un valor no válido se impide la realización del cálculo. Sin embargo sí es posible realizar el cálculo con advertencias. Los posibles errores y advertencias generados por cada parámetro se detallan a continuación.

Puntos de red

Para realizar el cálculo numérico el espacio se discretiza en una cantidad determinada de puntos de red. Cuanto mayor sea el número de puntos más lento será el cálculo pero mayor será la precisión obtenida. Para un desempeño óptimo recomendamos usar 1000 puntos de red para encontrar las soluciones y a continuación usar el comando “Refinar resultados” en la pantalla de resultados que mejora la precisión de los resultados repitiendo el cálculo con 10 veces más puntos de red.

Este parámetro debe ser entero y positivo

Momento angular

Para potenciales tridimensionales existe la posibilidad de que la partícula (o el sistema de partículas) tenga momento angular. Para potenciales unidimensionales el campo queda bloqueado ya que no tiene sentido la existencia de momento angular.

Este parámetro debe ser entero y no negativo

Salto de energía

Para hallar las soluciones de energía se recorre la distancia entre la energía inicial y la final en intervalos del tamaño indicado por el salto de energía. Si dos soluciones se encuentran dentro del mismo intervalo definido por el salto de energía no encontrará ninguna de las dos. Cuanto mayor sea el valor del salto de energía más rápidamente recorrerá el intervalo entre la energía inicial y final pero es posible que no encuentre soluciones si éstas se encuentran cercanas. Si al visualizar las ondas de las soluciones el número de nodos no aumenta uniformemente con cada nueva solución es probable que se hayan perdido soluciones y hay que repetir el cálculo con un paso menor.

Este parámetro debe ser no negativo

Energía inicial

Indica la energía inicial en la que se iniciará el cálculo. La energía inicial tiene que ser menor que la energía final y menor que la energía máxima para la cual existen estados ligados. Si la energía inicial es menor que el mínimo del potencial el cálculo se iniciará en el mínimo del potencial. Si la energía inicial es mayor que el mínimo del potencial obtendremos una advertencia de que puede haber soluciones por debajo de la energía inicial. La numeración de las soluciones irá marcada con el símbolo ~ ya que no podemos asegurar que realmente se haya encontrado el primer estado.

Este parámetro debe ser numérico, menor que la energía final y menor que la energía máxima para la cual existen estados ligados.

Energía final

Indica la energía final en la que se detendrá el cálculo. La energía final tiene que ser mayor que la energía inicial y mayor que la energía mínima del potencial. Si la energía final es mayor que el máximo del potencial el cálculo se detendrá en el máximo.

Este parámetro debe ser numérico, mayor que la energía inicial y mayor que el mínimo del potencial.

Número de soluciones

Determina el número máximo de soluciones que se calcularán. El cálculo se detendrá tras encontrar este número de soluciones o al alcanzar la energía final, lo que ocurra antes.

Este parámetro debe ser entero y positivo.
